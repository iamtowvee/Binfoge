# Binfoge

## About the extension

This extension is the official extension for the **Binfoge** programming language.

It is needed for syntax highlighting of `.upc` and `.bn` (and `.bnf`, respectively) files.

Version: `26.4.1` (Supported Binfoge language version: `A26.04.1`)

> [!NOTE]
> The language compiler is downloaded separately: [GitHub Releases](https://github.com/iamtowvee/binfoge/releases).

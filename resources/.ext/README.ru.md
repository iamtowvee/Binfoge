# Binfoge

## О расширении

Это расширение - официальное расширение для языка программирования **Binfoge**.

Нужно для подсветки синтаксиса файлов `.upc` и `.bn` (и `.bnf` соответственно).

Версия: `26.4.1` (Поддерживаемая версия языка Binfoge: `A26.04.1`)

> [!NOTE]
> Компилятор языка скачивается отдельно на: [GitHub Releases](https://github.com/iamtowvee/binfoge/releases).

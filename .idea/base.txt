Binfoge - высокоуровневый нативно-компилируемый язык программирования со статической типизацией.

Версия: 26.04.1A

cli:
(?) Аргументы: <arg> обязательный; [arg] опциональный

    binfoge --help
    binfoge --docs [lang]
        ^ LANG:
            * ru
            * en
    binfoge debug <path_to_file.bnf> [args]
        ^ ARGS:
            :save_log_to:"path_to_file"
    binfoge build <path_to_config> [args]
        ^ ARGS:
            :save_log_to="path_to_file"
            :save_c_sources
            :verbose
    binfoge test <path_to_config> [args]
        ^ ARGS:
            :save_log_to="path_to_file"
            :save_c_sources
            :verbose

Примеры команд:
    binfoge --docs ru
    binfoge build binfoge.upc :save_c_sources :save_log_to="logs/last.txt" :verbose

Процесс компиляции:

user_code -> binfoge.exe -> C -> gcc/mingw/clang/tcc -> EXE

Binfoge - сам язык программирования, расширение .bnf (или .bn, тоже поддерживатеся)
file.bnf:
```binfoge
fn int calc(str op, int first, int second) {
    case (op) {
        "+" => { return first + second; },
        "-" => { return first - second; },
        "*" => { return first * second; },
        "/" => {
            if (second != 0) {
                return first / second;
            } else {
                sendln!("Infinity.");
                return 0;
            }
        },
        ["^","**"] => { return first ** second; },
        "%" => { return first % second; },
        _ => {
            sendln!("Unknown.");
            return 0; 
        }
    }
    return 0; // Every function must have a return expression before its end
}

pub int main() {
    let str c = "+";
    let int a = 14;
    let int b = 28;

    let int x = calc(c, a, b);
    if (c == "/" && b == 0) {
        stmt::null;
    } else {
        sendln!(x);
    }

    return 0;
}
```

А так же
```
int anything() {
    const double y = 3.14;
    sendln!(y);
    return 0;
}

anything(); // создает y
anything(); // не создает y
```

Первая версия предусматривает:
sendln!(3.14 | " <-- it's pi " | 4 | 'c' | 'h' | 'a' | 'r' | true);
Где ! - не макрос, а обозначение "функция из core или std"
